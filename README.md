# PlonkJS

Javascript/NodeJS Plonk proving and verifiing library on [eigen-zkit](https://github.com/0xEigenLabs/eigen-zkit) and [bellman](https://github.com/matter-labs/bellman).

# Usage

## test
```
yarn build_node
yarn test
```

## compile
```
yarn build_node  #node
yarn build  #web
```
